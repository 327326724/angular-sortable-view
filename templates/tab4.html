Multiple multi-sortable lists

This example shows the usage of angular-sortable-view with multiple multi-sortable lists. The first column is connected with the third, and the second is connected with the fourth.

In order to connect sortable roots (elements with sv-root attribute) you need to provide the value of the sv-root attribute. It will be an identifier that groups them together. If no value for this attribute is provided then only parts (elements with sv-part attribute) with the common sv-root ancestor will be connected with each other. In case of this example the groups are identified by two identifiers: multiple_false and multiple_true (it just takes advantage of the ngRepeat's inner $odd property that alternates between true and false).
